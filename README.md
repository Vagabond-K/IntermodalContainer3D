# 그래픽 툴 없이 WPF로만 만든 3D 화물 컨테이너

순수하게 WPF의 레이아웃 기능과 2D 및 3D 그래픽 기능만 사용해서 만든 화물 컨테이너 샘플 프로젝트입니다. WPF의 레이아웃 기능을 사용했기 때문에 설정 값에 따라 WPF의 가변 레이아웃을 활용하여 텍스처 이미지를 유동적으로 생성할 수 있습니다.

자세한 사연은 [블로그 글](https://blog.naver.com/vagabond-k/224331074417)에 작성했으며, 다음의 그림과 영상은 실행 예시입니다.
